Входным параметром метода является одномерный массив строк.
Сначала метод вычисляет количество элементов входного массива больше 3 символов используя цикл, который поэлементно делает проверку содержит ли элемент больше трёх символов или нет. Это значение будет количеством элементов второго массива.
Затем создаём массив с вычисленным ранее значением количества элеметов и с помощью аналогичного цикла поэлементно делаем проверку содержит ли элемент больше трёх символов или нет, если нет, то копируем элемент в выходной массив.
